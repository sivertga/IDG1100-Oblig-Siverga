Section I – OS, Kernel, Shell

1. What components are part of the Von Neuman Architecture

A processing unit that contains an arithmetic logic unit and processor registers, A control unit that contains an instruction, register and program counter, Memory that stores data and instructions, External mass storage, Input and output mechanisms

2. What is user space?

User space is the location where the normal user processes run. These processes run with limited privileges, while the kernel space has access to the user sections, and runs in supervisor mode.

3. What is the shell?

A Shell provides you with an interface to the Unix system. It gathers input from you and executes programs based on that input. When a program finishes executing, it displays that program's output. There are many shell variants, for example Bourne shell, C shell, emacs shell.

4. Where is the shell located?

Inside the hardware-> kernel space.

5. What is the main task of the shell?

Perform and read tasks from your computer.

Section II – UNIX, Scripting

6. What is the octal representation of a file that can be read and written only by its owner?

The octal digit is 6. User got 4+2, group got 0, other got 0. So Chmod 600

7. In UNIX systems, what is the common name of the special user that can access and modify every file?

Superuser can access all files and edit them. When you’re a superuser, you operate in the root folder.

8. Which file permission classes are available in UNIX systems?

User, group, other, all.

9. As of 2020, which percentage of servers have UNIX-like OS?

About 70% uses an UNIX-like OS.

10. The command ssh will:

Login to a remote computer and perform operations. Can also be used to transfer data.

11. UNIX Command: access your current working directory:

You can use cd to navigate between directories, and you can view the directory you’re in by typing pwd.

12. UNIX Command: rename files (give an example) :

you can use mv to move the directory to another one, then you will be available to rename the file. Example: mv “startfile” “filenew”. This will get rid of the startfile and copy the content into the filenew directory.
